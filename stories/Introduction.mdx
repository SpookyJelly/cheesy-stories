Cheesy-stories is an add-on to Storybook that provides the ability to adjust the width of any story. This makes it easy for developers to see the changes in components that occur as the width changes.
Easy to use, just install and you are ready to go.

[![NPM](https://img.shields.io/badge/NPM-%23CB3837.svg?style=for-the-badge&logo=npm&logoColor=white)](https://www.npmjs.com/package/cheesy-stories)
[![Github](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white)][github-repo]

---

## Get started

1. Installation

```bash

npm install --save-dev cheesy-stories
```

```bash
yarn add -D cheesy-stories
```

2. Resister the addon in `.stories/main.ts`

```typescript
const config = {
  // ...
  addons: ["cheesy-stories"],
};
```
